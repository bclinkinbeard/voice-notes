# Redesign Team Proposals

Four focused design agents each explored three directions for the app. Every concept optimizes for quick capture, obvious common actions, and minimal default chrome for lower-frequency tasks.

## Agent 1 — Workflow Lead

1. **Signal Console** — Lead with the three highest-frequency capture actions, keep file/photo secondary, and make inbox scanning feel like triage.
2. **One-Tap Recorder** — Collapse everything except voice into a compact strip so the record action dominates the first screen.
3. **Command Stack** — Put ask, capture, and current work into a single stacked dashboard that supports rapid context switching.

**Implemented pick:** **Signal** — best balance of speed, clarity, and low visual noise.

## Agent 2 — Information Architect

1. **Ledger Notebook** — Use bright, structured surfaces and clearer section boundaries so entries read like reliable records.
2. **Digest View** — Add stronger summaries above the timeline so users can orient before scrolling.
3. **Split Library** — Separate projects, topics, and raw captures into tabs for users with larger vaults.

**Implemented pick:** **Ledger** — strongest reading comfort and trust for heavy text review.

## Agent 3 — Product Systems Designer

1. **Atlas Dashboard** — Put focus areas beside the timeline to make project and topic navigation faster.
2. **Board Mode** — Turn current projects into columns for a stronger task-management posture.
3. **Review Rail** — Create a persistent “next / waiting / due” strip above the stream for action planning.

**Implemented pick:** **Atlas** — most effective overview-first redesign without complicating capture.

## Agent 4 — Visual Restraint Specialist

1. **Nocturne Focus** — Sharpen contrast, reduce glow, and make a dark theme that feels serious instead of decorative.
2. **Mono Utility** — Remove most visual personality and rely on type, spacing, and borders only.
3. **Night Ledger** — Blend dark surfaces with paper-like hierarchy for quieter long-session review.

**Implemented pick:** **Nocturne** — best dark-mode option with minimal softness or gradient dependence.

## Implemented Variant Summary

- **Signal** — default fast-lane layout with sticky capture and accent-led action states.
- **Ledger** — clean light notebook theme for low-friction reading.
- **Atlas** — overview-first dashboard with wider project/topic visibility.
- **Nocturne** — compact high-contrast dark variant for focused sessions.
